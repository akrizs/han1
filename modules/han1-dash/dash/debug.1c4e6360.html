Definition of each OBIS Group

OBIS Group Code Definition
A Specifies the medium (0=abstract objects, 1=electricity, 6=heat, 7=gas, 8=water etc.)
B Specifies the channel. Each device with multiple channels generating measurement results, can separate the results into the channels
C Specifies the physical value (current, voltage, energy, level, temperature, etc.)
D Specifies the quantity computation result of specific algorythm
E Specifies the measurement type defined by groups A to D into individual measurements (e.g. switching ranges)
F Separates the results partly defined by groups A to E. The typical usage is the specification of individual time ranges,
