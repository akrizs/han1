Welcome To HAN1

Please follow the setup to get you up and running with your AMS/HAN enabled smartmeter!

HAN1 is an NodeJS project to read, parse and serve data from so called "smart electrical meters", the system works after the Norwegian HAN standard which follows the DLMS/COSEM standard.

General

Few basic generic settings to get you connected to the meter and start receiving data from it!

Meter

Few settings to get the meter syncronized with the HAN1 system!

Dashboard

Settings for the dashboard and the web stack services.

Addons

Addons also have some settings to be adjusted!

Storage

You can provide storage options for the system to save the values to be read from the meter, a storage option is mandatory if you want to view historic data and go back in time to find readings!

Review your settings!

Please go over the settings you have choosed to prevent any errors at first startup.
